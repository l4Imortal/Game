# 🎮 Jogo Roblox - Projeto em Desenvolvimento

Este repositório contém o desenvolvimento do meu jogo criado na plataforma **Roblox**.
Aqui eu organizo os arquivos, scripts e recursos utilizados durante a criação do jogo.

## 📌 Sobre o Projeto

Este projeto foi criado com o objetivo de aprender mais sobre desenvolvimento de jogos, lógica de programação e criação de experiências interativas dentro do Roblox.

O jogo está atualmente **em desenvolvimento**, então novas atualizações e melhorias serão adicionadas ao longo do tempo.

## 🛠️ Tecnologias Utilizadas

* **Lua** – linguagem usada para scripts no Roblox
* **Roblox Studio** – ferramenta de desenvolvimento de jogos da plataforma
* **Git & GitHub** – controle de versão do projeto

## 📂 Estrutura do Repositório

* `scripts/` → Scripts do jogo
* `assets/` → Imagens, modelos e outros recursos
* `docs/` → Documentação e anotações do projeto

## 🚀 Objetivo

Meu objetivo com esse projeto é:

* Aprender desenvolvimento de jogos
* Melhorar minhas habilidades em programação
* Criar experiências interativas dentro do Roblox
